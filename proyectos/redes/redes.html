Redes

Proyecto de configuración y simulación de redes

Este proyecto consiste en el diseño, configuración y simulación de redes utilizando herramientas como Packet Tracer.

🌐 Configuración de redes + troubleshooting

Se implementaron topologías de red, direccionamiento IP, configuración de routers y switches, además de pruebas de conectividad para asegurar el correcto funcionamiento.

🖥️ Topología de red

Diseño de la topología de red implementada en el proyecto.

⚙️ Configuración de dispositivos

Configuración de routers y switches, incluyendo direccionamiento IP y protocolos básicos.

📡 Pruebas de conectividad

Verificación del correcto funcionamiento de la red mediante pruebas de comunicación entre dispositivos.
