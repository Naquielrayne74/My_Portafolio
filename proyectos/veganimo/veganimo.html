Veganimo

Aplicación web enfocada en alimentación saludable

Este proyecto consiste en una aplicación web orientada a la promoción de alimentación saludable y recetas veganas.

🥗 Backend + Frontend

Se desarrollaron funcionalidades para mostrar recetas, gestionar información nutricional y mejorar la experiencia del usuario mediante una interfaz amigable.

Tecnologías utilizadas: Visual Studio Code, MongoDB Atlas, JavaScript y PHP.

🍽️ Vista principal

Interfaz principal donde el usuario puede iniciar sesión, registrarse y explorar la información de la página web.

🥗 Suscripción a planes alimenticios

Sección donde el usuario puede suscribirse y acceder a planes alimenticios con pasos estructurados para una alimentación saludable.

🔐 Inicio de sesión

Pantalla de autenticación donde los usuarios pueden iniciar sesión para acceder a sus datos y funcionalidades personalizadas.
