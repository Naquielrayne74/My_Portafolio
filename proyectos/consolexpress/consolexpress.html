ConsoleXpress

Aplicación web de venta de consolas

Este proyecto consiste en una aplicación web enfocada en la venta de consolas de videojuegos.

💻 Backend principal + apoyo en Frontend

Me encargué de la lógica del servidor, manejo de datos y estructura del sistema, asegurando un funcionamiento eficiente y seguro. Además, colaboré en la interfaz para mejorar la experiencia del usuario.

Tecnologías utilizadas: Python, MySQL, phpMyAdmin y Visual Studio Code.

🛒 Vista principal

Interfaz principal donde el usuario puede explorar

📦 Catálogo de consolas

Listado dinámico de consolas con opciones de exploración y selección.

🤖 Chatbot de asistencia

Chatbot de asistencia integrado que responde preguntas sobre consolas, ayudando al usuario a resolver dudas y navegar el catálogo de productos.

🧾 Proceso de compra

Gestión del carrito y proceso de compra del usuario dentro de la aplicación.
